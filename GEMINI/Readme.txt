# GEMINI AI

GEMINI AI is a Streamlit-based chat interface for interacting with the GenerativeAI model.

## Setup

1. **Install Dependencies:**

   ```bash
   pip install -r requirements.txt


2. Paste your API Key:

Open api.json and paste your GenerativeAI API key:

{
  "api": "YOUR_GENERATIVEAI_API_KEY"
}

3. Run the Application:

    streamlit run your_app_file.py

Configuration
Make sure to configure the api.json file with your GenerativeAI API key.

Logging
The application logs messages, user inputs, assistant responses, and errors to the logs/chat_log.txt file.
